# Portfólio Pessoal

Este projeto foi desenvolvido como atividade prática da disciplina de Fundamentos da Programação Web.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Página Sobre Mim
- Página Formação
- Página Portfólio
- Página Contato
- Formulário com validação em JavaScript
- Simulação de envio
- Alternância entre tema claro e escuro
- Layout responsivo
